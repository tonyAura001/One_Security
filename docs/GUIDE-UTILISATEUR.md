# PilotePME — Guide utilisateur

Chaque utilisateur ne voit que les écrans autorisés par son rôle. Les boutons d'action
n'apparaissent que si votre rôle a le droit d'écrire.

## Connexion
Rendez-vous sur l'adresse de la plateforme, saisissez votre e-mail et mot de passe.
Le bouton **Déconnexion** est dans le menu de profil (en bas du menu latéral).

## Direction (DG)
- **Tableau de bord** : santé financière (CA, recouvrement, masse salariale), courbe de
  trésorerie, **CA boutique 14 jours**, répartition des déploiements, indicateurs opérationnels.
- **Voir en tant que** : visualisez l'interface d'un autre rôle (vos droits data restent inchangés).
- Accès à tous les modules.

## Finance & Comptabilité (RF / Comptable)
- **Facturation** : créer une facture, suivre les statuts (émise → envoyée → payée / en retard).
- **Trésorerie** : mouvements par compte, courbe de solde, **Exporter** (Excel/CSV).
- **Relances** : relancer les factures en retard.
- **Fournisseurs / Contrats** : gestion + export.
- **Caisse/Boutique** : catalogue produits, ventes (POS), reçus.

## Opérations (RP / Manager / Contrôleur)
- **Agents** : fiche des 67 agents (identité, matricule, poste). Édition réservée aux responsables. **Exporter** disponible.
- **Planning** : affecter une vacation (jour/nuit/renfort) à un agent sur un site.
- **Présences / Pointage** : suivi des pointages.
- **Incidents** : déclarer un incident (type normalisé : agression, intrusion, vol…).
- **Maintenance** :
  - *Tickets* (kanban par criticité) — glissez une carte pour changer d'étape.
  - *Maintenance préventive* — créez un **plan** (périodicité + échéance) ; le bouton **Générer le ticket** crée un ticket et repousse l'échéance. Génération aussi automatique chaque nuit.
  - *Interventions* — comptes rendus.

## RH (RH / Manager)
- **Recrutement** : postes, candidatures, entretiens, pièces jointes (CV).
- **Paie** : bulletins, export paie.

## Projets / Déploiements
- Liste des déploiements (avancement, budget, statut) + **Exporter**.
- Fiche déploiement : synthèse, **Documents du déploiement** (pièces jointes), **fil de commentaires**.

## Community Management (RP / RH / Manager)
- **Plateformes** : déclarez vos réseaux gérés (activer/désactiver au clic).
- **Calendrier éditorial** : planifiez des publications ; bouton **Engagement** pour saisir vues/j'aime/commentaires/partages/clics (le total se met à jour).

## Documents (DG / RP / RF / RH / Comptable / Manager)
- Créez un **devis, facture proforma, rapport, fiche d'engagement ou communiqué** (charte One Security).
- Éditez les champs → aperçu A4 en direct → **Exporter PDF** (impression).
- **Signature** : nom, fonction, date + tracé à la souris/au doigt, apposée sur le document.
- **Historique** : chaque enregistrement crée une **version** ; bouton **Restaurer** pour revenir en arrière.

## Communs à tous
- **Messagerie** : canaux de groupe + **messages directs** privés (bouton « Nouveau message direct »).
- **Tâches** : vos tâches assignées/créées.
- **Exporter** : présent sur les principaux tableaux (télécharge un fichier Excel/CSV des lignes filtrées).
